# Route 53とは
https://aws.amazon.com/jp/route53/

AWSが提供するDNS(ドメインネームサービス)になります。  
公式のこの一言が直感的で分かりやすそうです。

> www.example.com のような名前を、コンピュータが互いに接続するための数字の IP アドレス (192.0.2.1 など) に変換するサービス



```
memo: DNSが使用するポート番号が53
```


# 料金
https://aws.amazon.com/jp/route53/pricing/

- 0.50USD ホストゾーンごと / 月、最初の 25 ホストゾーン
- 0.40USD 100 万クエリごと – 最初の 10 億クエリ / 月

ホストゾーン１つが１つのドメインを管理するグループの単位となります。  
毎月1ドルはいかないような値段となっています。
