## tfenv導入

rbenvやpyenvのノリで  
terraformにもversion管理のツールがあるのでそちらを使います。

macでの導入。
```shell:terminal
brew install tfenv
```

以下のコマンドでterraform最新versionを使用することができます。
```shell:terminal
tfenv install latest
tfenv use latest
```
