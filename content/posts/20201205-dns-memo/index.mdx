# DNS

https://aws.amazon.com/jp/route53/what-is-dns/

Domain Name System (!= Server)

> www.example.com のような名前を、コンピュータが互いに接続するための数字の IP アドレス (192.0.2.1 など) に変換するサービス

192.0.2.1のようなIPアドレスはサーバーの住所。
人が覚えるには大変なので、名前を付けてDNSを用いて名前解決をします。


## ネームサーバ 

権威DNSサーバ, コンテンツDNSサーバ

ドメインとIPアドレスを紐つける台帳をもったサーバ



## フルリゾルバ

DNSキャッシュサーバ

ネームサーバに聞きに行ってくれる役割。


## Route53
Route53はネームサーバを提供する。

自身でこのドメインはこのIPアドレス みたいな情報を追加していく。


### SOAレコード

ドメインの管理情報を示すレコード

- SOAレコードを作成したホスト
- リフレッシュ時間
- キャッシュ時間

### NSレコード

ドメインのネームサーバを指定しているレコード
以下でドメインに紐つくネームサーバを見ることができます。

```shell
dig [domain name] ns
```
